About the plugin

Usage

Add the npm package to your project:

# Via yarn
$ yarn add serverless-dynamodb-ttl

# Via npm
$ npm install serverless-dynamodb-ttl --save

Add the plugin to your serverless.yml:

plugins:
  - serverless-dynamodb-ttl

Configure TTL in serverless.yml:

custom:
  dynamodb:
    ttl:
      - table: your-dynamodb-table-name
        field: your-ttl-property-name

That‚Äôs it! After the next deployment (sls deploy) serverless will configure your TTL properties in DynamoDB.
