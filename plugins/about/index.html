Hello!

Serverless Directory was created to help people find plugins more easily.

I was inspired by a few great websites that exist to help developers be more efficient.

  • The late Ruby Toolbox always gave a super quick way to browse through plugins by category.

FAQ

How do I add a Plugin to the listing?
Visit github.com/serverless/plugins and add your repo to the list. We sync up with it.
My plugin is listed here, how do I add topics?
Add Topics to your GitHub Repo to have them added here.
