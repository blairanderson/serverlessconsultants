## Usage

```
plugins:
  - serverless-plugin-typetalk

custom:
  typetalk:
    token: "XXXXXXXXX"
    topicId: "123456"
```

For working example, see [this repository](https://github.com/is2ei/serverless-plugin-typetalk-example)
